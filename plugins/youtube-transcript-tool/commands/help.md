Display help information for youtube-transcript-tool CLI commands.

## Usage

```bash
# Show general help
youtube-transcript-tool --help

# Show command-specific help
youtube-transcript-tool COMMAND --help

# Show version
youtube-transcript-tool --version
```

## Arguments

- `COMMAND` (optional): Specific command to get help for
- `--help` / `-h`: Show help information
- `--version` / `-v`: Show version information

## Examples

```bash
# General help
youtube-transcript-tool --help

# Command help
youtube-transcript-tool search --help

# Version information
youtube-transcript-tool --version
```

## Output

Displays usage information, available commands, and options.
